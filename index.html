Wat ik doe

Ik design en develop websites voor mensen die een website nodig hebben. Ik kan websites verbeteren van mensen die misschien een hele oude website hebben, maar ik kan ook natuurlijk gloednieuwe websites maken. Dit allemaal voor een erg lage prijs, aangezien websites maken een grote hobby van mij is.

Waarom jij?

Iedereen die een bedrijf heeft, heeft wel een website nodig. In deze tijd is iedereen er aan gewend om een website van een bedrijf te bekijken voor meer informatie. Zelfs mensen zonder bedrijf kunnen natuurlijk ook een website nodig hebben. Dus heeft u nu een oude website, of nog helemaal geen een, bij Kroesweb kunt u een mooie website voor een lage prijs aanschaffen.
